# Botão
O componente de botão é um elemento fundamental da interface do usuário, presente em aplicações e websites para permitir a interação dos usuários com o sistema. Ele é representado graficamente por uma área retangular que pode conter texto, ícones ou ambos, e é projetado para responder ao clique do usuário.
