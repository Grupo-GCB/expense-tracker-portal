# Error Page
Essa página é uma página de erro para ser exibida quando ocorrer um erro de servidor (código de erro 500) durante uma requisição ou processamento de dados. O layout da página é responsivo e ajusta-se ao tamanho da tela do dispositivo, garantindo uma boa experiência do usuário em diferentes dispositivos e resoluções.
